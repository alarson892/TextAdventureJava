# Commands

The game will be played by typing commands at the command prompt.  
The prompts will be numbers.
If a room doesnt allow a command, then that command will be printed in red.
Entering an invalid command will display an error message.

## Commands

    1) Move
        1) left
        2) up
        3) down
        4) right
    2) Pickup item
        1) left hand
            1) from floor
            2) from wall
            3) from backpack
            4) from table
            5) from right hand
        2) right hand
            1) from floor
            2) from wall
            3) from backpack
            4) from table
            5) from left hand
    3) Place item
        1) left hand
            1) to floor
            2) to wall
            3) to backpack
            4) to table
            5) to right hand
        2) right hand
            1) to floor
            2) to wall
            3) to backpack
            4) to table
            5) to left hand
    4) Use item
        1) left hand
        2) right hand
    5) Fight
    6) Run away
    7) Stats
    8) Look Around
    9) Game Options
        0) Help
        1) New Game
        2) Save Game
        3) Quit Game
